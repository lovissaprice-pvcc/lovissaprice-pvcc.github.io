Term Definition
Machine Language The language understood by a computer
Debugging Identifying and removing errors from computer hardware or software
Assembler Converts mnemonic expressions into machine language instructions
Assembly Language Mnemonic systems for representing programs
Translator Translates programs expressed in high-level primitives into machine-language programs
Compiler Compiles several machine instructions into short sequences to simulate the activity requested by a single high-level primitive
Interpreter Executed a program from its high-level form
Cross-Platform Software Independent of an operating system’s design as well as the machine’s hardware design
Natural Language English, German, Viet, etc.
Formal Language Precisely defined by grammars (programming languages)
Machine Independence Software that can be used on any machine
High-Level Programming Language Provide more abstractions for the programmer and make it easier for people to read and write a program
Object-Oriented Programming (OOP) A collection of units called "objects" which each complete tasks related to themselves and request other objects to complete tasks
Scripting Language Performs administrative tasks rather than to develop complex programs
Data Types:
  1. Integer
  2. Float
  3. String
  1. Numeric data consisting of whole numbers
  2. Numeric data that might contain values other than whole numbers
  3. Data that is not whole numbers, such as a word or phrase
Primitive Data Type Data types that are included as primitives in a programming language, such as for integer and for character
Data Structure Conceptual shape or arrangement of data
Python Collection Data Structures:
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. A list of data
  2. A row in a relation
  3. Used to store multiple items in a single variable
  4. Used to store data values in key:value pairs
Function A block of code which only runs when it is called
Local Variable Variables that are created inside of a function
Global Variable Variables that are created outside of a function
Source Program Instructions that guide the program's execution
Object Program A program into which some other program is translated by an assembler or compiler
