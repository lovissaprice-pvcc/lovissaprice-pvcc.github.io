Definition

  1. software packages that coordinate a computer’s internal activities as well as oversee its communication with the outside world
  2. a list of tasks to be carried out in turn by an operating system
  3. first-in, first-out
  4. allows a program being executed to carry on a dialogue with the user through remote terminals
  5. method of operation in which multiple users with different programs interact nearly simultaneously with the CPU
  6. time is divided into intervals and then the execution of each job is restricted to only one interval at a time
  7. handles the communication with the user of the databas
  8. a program designed to run a computer's hardware and applications and manage its resources
  9. software units that extend (or perhaps customize) the capabilities of the operating system
  10. allow users to issue commands by using one of several common input devices
  11. communicated with users through textual messages using a keyboard and monitor screen
  12. the internal part of an operating system
  13. the software units that communicate with the controllers
  14. coordinate the use of the machine’s mass storage facilities
  15. coordinating the machine’s use of main memory
  16. imaginary memory space whose apparent presence is created by the process of swapping data and programs back and forth between main memory and mass storage
  17. transfers the operating system from mass storage (where it is permanently stored) into main memory
  18. contains a collection of software routines for performing fundamental input/output activities
  19. two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
  20. holding data for output at a later but more convenient time
  21. gains highly privileged access to the operating system by identifying him- or herself as the administrator
  22. requires more than one piece of evidence to gain access
